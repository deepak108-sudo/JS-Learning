# Quotes Generator

A simple JavaScript-based Quotes Generator that displays random motivational quotes every 2 seconds.

## Features
- Randomly displays quotes from a predefined list of 30 quotes.
- Shows the author's name.
- Styled quote box with border and shadow.
- Automatically updates quote every 2 seconds.

## Demo Screenshot
![Quotes Generator Screenshot](images/screenshot.png)

## How to Run
1. Clone or download the repository.
2. Make sure you have the following files:
   - `index.html`
   - `first.js` (your JavaScript file)
3. Open `index.html` in a web browser.
4. The quote generator should automatically display random quotes every 2 seconds.
